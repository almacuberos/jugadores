### Librerias 

- streamlit
- pandas

### Ejecución en Windows
			
#### La primera vez

* Bajar fuentes

git clone https://github.com/almacuberos/jugadores.git

* Crear environment

cd jugadores

cd pythonProject

python -m venv venv

* Activar environment

venv\Scripts\activate

* Instalar librerias

pip install -r requirements.txt

#### El resto de las veces 

cd jugadores

cd pythonProject

venv\Scripts\activate

#### Para ejecutar 

streamlit run jugadores.py
